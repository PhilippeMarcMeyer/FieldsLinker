FieldsLinker
 v 0.95 : rewritten for multiples instances in mind
New option for touch devices {"mobileClickIt":true} : idea by Norman Tomlins , make links more easily on touch devices just by clicking

Allows to link elements of list A to elements of list B and returns the links in a js object

  • The elements of a list are sortable by drag and drop for a better lisibility
  • Colours can be re-defined and links come in 2 flavours
    1. Straight lines
    2. Lines with squares at the ends
  • You may input somme links from a previous session
  • Mandatory fields (optional array)
  • Works in Chrome, Chromium, Opera, Firefox and IE (9+)
  • Result in an object with error true/false plus an array of links
  • Actions are : init, eraseLinks, getLinks, changeParameters, disable, enable
